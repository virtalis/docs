Dynamic Sections

A Dynamic Section is a plane that can be used to clip geometry in your scenes. It removes all the geometry to one side of the plane and can be applied to standard geometry, point clouds and scene GUI. They permit designers to visually inspect sections of their products such as the internals of a wind turbine.

note

Visionary Render can support up to 6 dynamic sections at once.

Create​

To create a Dynamic Section, right-click on the Scenes node in the Data window and select Create > Dynamic Section from the context menu. This will create a Dynamic Section node beneath a Dynamic Section List called Sections.

You can organise several Dynamic Sections in your scene beneath this list. Dynamic Sections can also be parented to Assemblies. To do this, right-click on an Assembly and select Create > Dynamic Section. When a Dynamic Section is parented to an Assembly it will only clip the parent Assembly and its children.

Properties​

PropertyDescription
EnabledSimple flag to indicate whether a section is enabled. [default: true]
VisibleDraw a plane in the scene to indicate the position and orientation of the section. [default: true]
Enable CapsFill in the gaps in geometry caused by the section clipping it (i.e. cap it off). [default: false]
AlignmentThe initial plane alignment of the section. There are 4 options: XY, XZ, YZ and Custom. These just represent default values – the orientation of the section can be changed to a custom value from the properties of the parent Assembly (in which case, the Custom option will be automatically set). [default: XY]
InvertSwap the side of the plane to clip geometry from. [default: false]
Section LinkThe Assembly to apply the section to (including its children). Typically this will be the Scenes node so that the section is applied to the whole scene. [default: parent node]
OpacityThe transparency of the section plane if the Visible property is enabled. [default: 0.5]

Settings​

Dynamic sections can be configured via Settings > Dynamic Sections.

SettingDescription
ColourThe colour to apply to section planes. [default: white]
Selection ModeThe selection behaviour of section planes. There are 3 options:Disable – prevent section planes for being selectable.Plane – click anywhere on a section plane to select it.Edge of Plane – click near the edge of a section plane to select it. [default]
Assign To Scene RootCreate sections with their Section Link property set to the Scenes node (to apply the section to the entire scene). Otherwise, new sections will be applied to their parent Assemblies. [default: false]
Size on DropWhen enabled, moving sections will not update their visual bounds until the movement is complete. This can improve performance when clipping complex Assemblies by reducing the number of sizing calculations performed during dragging. [default: true]
OpacityThe default opacity to apply to new sections. [default: 0.5]
