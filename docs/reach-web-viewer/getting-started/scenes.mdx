# Scenes

After clicking on the ‘Beta’ button, you will see the Scenes page (note that it may have either light or dark colours depending on your operating system’s colour mode). It will show the same list of scenes as the official version of Reach. Left-click on a scene to open it.

## Search

Left-click in the search bar and start typing to search for scenes by name.

:::note
It is case-insensitive and does not support wildcards.
:::
