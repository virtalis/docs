# 2024.3.1

GeoVisionary 2024.3.1 is a hotfix release addressing a critical issue for users working with large datasets. Below is the summary of the update included in this version.

## Resolved Issues

| ID | Summary
|---|---|
| FUS-3580 | Fixes a read error when loading files containing more than 4GB of XML or mesh data in a single block. |

This release focuses on enhancing the robustness of data handling for large files, ensuring better stability for complex workflows.
