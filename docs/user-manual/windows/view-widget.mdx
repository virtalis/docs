# View Widget

The View Widget shows the current camera position and orientation in the scene and permits these values to be changed. It is rendered to a transparent frame over the 3D scene.

![](/img/user-manual/chapter9/1.jpg)

To open the View Widget, click on the Virtalis button at the top-left of the Toolbar and select **Windows** \> **View Widget**. Alternatively, click and hold left mouse button over the Windows menu of the Toolbar and select **View Widget**. It can also be shown or hidden with **Ctrl** + **U**.

To close the View Widget, right-click on it near the axes and select **Close** from the context menu. To move it, hover the mouse over the axes so that the movement cursor appears then click and drag.

## Axes

The axes rotate to reflect the current orientation of the camera. The x-axis is red, the y-axis is blue, and the z-axis is green.

## Buttons

### Viewpoints

These buttons provide quick access to 6 common perspectives. They give focus to the selected assemblies, or to the whole scene if nothing is selected.

| ![](/img/user-manual/chapter9/2.jpg) | Front view (`Alt` + `5`). | ![](/img/user-manual/chapter9/3.jpg) | Back view (`Alt` + `6`).   |
| ![](/img/user-manual/chapter9/4.jpg) | Top view (`Alt` + `3`).   | ![](/img/user-manual/chapter9/5.jpg) | Bottom view (`Alt` + `4`). |
| ![](/img/user-manual/chapter9/6.jpg) | Left view (`Alt` + `1`).  | ![](/img/user-manual/chapter9/7.jpg) | Right view (`Alt` + `2`).  |

### Misc.

The widget also contains buttons for resetting the orientation of the camera and changing the type of view.

|                          |                                                                              |
|--------------------------|------------------------------------------------------------------------------|
| ![](/img/user-manual/chapter9/8.jpg)  | Reset camera orientation to 0, 0, 0 (`Alt` + `7`).                           |
| ![](/img/user-manual/chapter9/9.jpg)  | Reset camera pitch to zero (`Alt` + `8`).                                    |
| ![](/img/user-manual/chapter9/10.jpg) | Reset camera roll to zero (`Alt` + `9`).                                     |
| ![](/img/user-manual/chapter9/11.jpg) | Switch between perspective and orthographic view. *\[default: perspective\]* |

## Number Edit Boxes

There are also number edit boxes that show the current position (x, y, z) and orientation (pitch, yaw, roll) of the camera. Click in one of the edit boxes to change its value.
