# Logging Flags

Flags that control the level of API error reporting. These can be combined. 

## Attributes

| Type           | Name           |
| -------------- | -------------- |
| const uint32_t | **[VRTREE_ERRORLEVEL_DEBUG](#vrtree-errorlevel-debug)** <br />Record debug messages.  |
| const uint32_t | **[VRTREE_ERRORLEVEL_ERRORS](#vrtree-errorlevel-errors)** <br />Record critial errors.  |
| const uint32_t | **[VRTREE_ERRORLEVEL_INFO](#vrtree-errorlevel-info)** <br />Record info messages.  |
| const uint32_t | **[VRTREE_ERRORLEVEL_NONE](#vrtree-errorlevel-none)** <br />Do not set any internal error codes nor generate any messages.  |
| const uint32_t | **[VRTREE_ERRORLEVEL_WARNINGS](#vrtree-errorlevel-warnings)** <br />Record warnings.  |

## Attributes Documentation

### VRTREE_ERRORLEVEL_DEBUG {#vrtree-errorlevel-debug}

```cpp
const uint32_t VRTREE_ERRORLEVEL_DEBUG = (1 << 2);
```

Record debug messages. 

### VRTREE_ERRORLEVEL_ERRORS {#vrtree-errorlevel-errors}

```cpp
const uint32_t VRTREE_ERRORLEVEL_ERRORS = (1 << 0);
```

Record critial errors. 

### VRTREE_ERRORLEVEL_INFO {#vrtree-errorlevel-info}

```cpp
const uint32_t VRTREE_ERRORLEVEL_INFO = (1 << 3);
```

Record info messages. 

### VRTREE_ERRORLEVEL_NONE {#vrtree-errorlevel-none}

```cpp
const uint32_t VRTREE_ERRORLEVEL_NONE = 0;
```

Do not set any internal error codes nor generate any messages. 

### VRTREE_ERRORLEVEL_WARNINGS {#vrtree-errorlevel-warnings}

```cpp
const uint32_t VRTREE_ERRORLEVEL_WARNINGS = (1 << 1);
```

Record warnings.
