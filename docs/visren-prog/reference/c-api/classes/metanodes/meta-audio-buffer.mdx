## Detailed Description

```cpp
struct Meta_AudioBuffer;
```

An AudioBuffer represents sound data that can be played by multiple sources. 

This node is usually created in a library and referenced from [Meta_AudioSound](../metanodes/meta-audio-sound.mdx) or [Meta_AudioTrack](../metanodes/meta-audio-track.mdx) nodes. These are positional.
