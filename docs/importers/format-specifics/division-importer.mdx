# Division Importer

The Division importer includes various format-specific settings.

## Entity Filers

| Setting            | Description                                         |
|--------------------|-----------------------------------------------------|
| Ignore Sequences   | Ignore sequences in the import. *[default: false]*  |
| Ignore Collisions  | Ignore collisions in the import. *[default: false]* |
| Ignore Events      | Ignore events in the import. *[default: false]*     |
| Ignore Constraints | Ignore constraints in the import. *[default: true]* |

## Reader Setup

| Setting            | Description                                                                              |
|--------------------|------------------------------------------------------------------------------------------|
| Always Load Models | What to do with assemblies when their load flag is disabled. If this option is enabled, the visuals of assemblies will be imported but initially disabled. If this option is disabled, the visuals of assemblies will not be imported to save memory and speed up the import. *[default: true]* |
| DIVISIONROOT       | Directory in which Mockup is installed on your hard drive. *[default: none]* |
| DIVISIONPATH       | Directory in which Mockup resources are installed on your hard drive. *[default: none]* |
