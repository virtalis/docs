# Introduction

This guide will show you how to setup and configure Virtalis Hub using shell commands. Virtalis Hub is a configurable platform composed of many interconnected microservices. This permits it to be adapted for different use-cases and environments.

:::note
This guide is aimed at a technical audience and requires prior knowledge of system administration and shell commands. To learn about general usage, see the [Virtalis Hub and Web Viewer Manual](../virtalis-hub-and-web-viewer/introduction.mdx).
:::
