# Immersive Context Menu

The Immersive Context Menu permits quick access to operations that you might want to perform whilst immersed. It can be opened using your HMD controller or via **Alt** + **Right-click** on a desktop computer.

![](/img/visionary-render/tracking/immersive-context-menu/icons.webp)

When you first open the menu you will see two icons as in the image above - a hamburger to show/hide the main menu and a wrench to show/hide the tool buttons.

:::tip
Both menus can be shown at once if you wish.
:::

## Main Menu

The main menu provides typical context menu options like undo/redo, cut, copy, paste, etc.

![](/img/visionary-render/tracking/immersive-context-menu/main-menu.webp)

The menu has 5 categories:

| Button | Description |
|--------|-------------|
| ![](/img/visionary-render/tracking/immersive-context-menu/edit.webp) | This page shows edit controls such as undo/redo, cut, copy, paste, etc. |
| ![](/img/visionary-render/tracking/immersive-context-menu/create.webp) | This page permits you to create basic geometric shapes and viewpoints. |
| ![](/img/visionary-render/tracking/immersive-context-menu/materials.webp) | This page shows all the materials in your scene. |
| ![](/img/visionary-render/tracking/immersive-context-menu/viewpoints.webp) | This page shows all the viewpoints in your scene. |
| ![](/img/visionary-render/tracking/immersive-context-menu/manikins.webp) | This page shows all the manikins in your scene and permits you to control manikins (Note: Immersive license required). |

## Tool Buttons

The tool buttons provide quick access to some of the tools in the Toolbar that you might want to operate whilst immersed.

![](/img/visionary-render/tracking/immersive-context-menu/tools.webp)

| Button | Description |
|--------|-------------|
| ![](/img/visionary-render/tracking/immersive-context-menu/select.webp) | Region select intersected or encapsulated objects in the scene with left-click + drag (**Ctrl** + **1**). This is enabled by default. Drag from left to right to select encapsulated objects, and from right to left to select intersected objects. When the toolbar is narrow, this is combined with the measuring and annotation tools in a drop-down (Note: only one of these can be activated at a time). |
| ![](/img/visionary-render/tracking/immersive-context-menu/measure.webp) | Measure distances between points by clicking on objects in the scene (**Ctrl** + **2**). |
| ![](/img/visionary-render/tracking/immersive-context-menu/annotate.webp) | Annotate objects in the scene by clicking on them (**Ctrl** + **3**). |
| ![](/img/visionary-render/tracking/immersive-context-menu/prs.webp) | Show the PRS manipulator to move, rotate and scale selected objects in the scene (**F9**). The drop-down menu permits the manipulators to be shown individually. |
| ![](/img/visionary-render/tracking/immersive-context-menu/helpers.webp) | Show helpers (for lights and particle systems, etc.). |
| ![](/img/visionary-render/tracking/immersive-context-menu/snap-home.webp)  | Snap selected objects in the scene when dragged near a previous transform (**F4**). |
| ![](/img/visionary-render/tracking/immersive-context-menu/avatar-shadows.webp) | Show avatar shadows. |
| ![](/img/visionary-render/tracking/immersive-context-menu/level-camera.webp) | Keep the camera level when tracking is enabled. |
| ![](/img/visionary-render/tracking/immersive-context-menu/laser.webp) | Show laser pointers on tracked controllers. |
