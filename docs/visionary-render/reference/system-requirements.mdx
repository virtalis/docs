# System Requirements

## Minimum

| Component         | Requirements                                                                                                                |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------|
| Operating system  | Microsoft Windows 10 Pro, 64-bit                                                                                            |
| CPU               | 4 processor cores (Intel Xeon, Intel Core i5, AMD Ryzen or equivalent)                                                      |
| Memory            | 8GB RAM                                                                                                                     |
| Hard drive        | 128GB HDD                                                                                                                   |
| Graphics hardware | OpenGL 3.3 compatible graphics card (NVIDIA GeForce 8000, NVIDIA Quadro FX 370, AMD Radeon HD, Intel HD 4000 or equivalent) |

## Recommended

| Component         | Requirements                                                                              |
|-------------------|-------------------------------------------------------------------------------------------|
| Operating system  | Microsoft Windows 10 Pro, 64-bit                                                          |
| CPU               | 8 processor cores (Intel Xeon, Intel Core i7, AMD Ryzen or equivalent)                    |
| Memory            | 64GB RAM                                                                                  |
| Hard drive        | 256GB SSD                                                                                 |
| Graphics hardware | NVIDIA Quadro P Series, NVIDIA GeForce RTX, AMD RX Vega or equivalent (8GB VRAM or above) |

## High-end

| Component         | Requirements                                                                                                          |
|-------------------|-----------------------------------------------------------------------------------------------------------------------|
| Operating system  | Microsoft Windows 10 Pro, 64-bit                                                                                      |
| CPU               | 10 processor cores or above (Intel Xeon, Intel Core i9, AMD Ryzen or equivalent)                                      |
| Memory            | 256GB RAM                                                                                                             |
| Hard drive        | 1TB SSD                                                                                                               |
| Graphics hardware | Dual GPU configuration (NVIDIA Quadro P Series, NVIDIA GeForce RTX, AMD RX Vega or equivalent [11GB VRAM or above]) |

## Cluster

If you plan to use Visionary Render on a cluster, there are some additional requirements:

- Each machine should meet the Recommended or High-end system requirements
- 10Gbit NIC in each machine
- Enterprise-grade network switch
- Graphics sync hardware in each machine (NVIDIA G-SYNC or equivalent)

## Tracking

Visionary Render supports the following tracking devices:

- Oculus Rift or Rift S, Meta Quest 2, tracking, software version 1.3 or above
- HTC Vive or Vive Pro HMD, Lighthouse tracking, pointers, SteamVR version 1.0 or above
- zSpace, software version 3.1 or above
- Windows Mixed Reality HMD, Windows MR Portal, SteamVR version 1.0 or above
- Microsoft HoloLens 2

:::note
These devices should be used with a machine that meets the Recommended or High-end system requirements.
:::
