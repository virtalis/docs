# Meta_ChromaKeyTexture

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

A ChromaKey texture can beused to change a colour in a [Meta_Movie](../metanodes/meta-movie.mdx) or [Meta_Texture](../metanodes/meta-texture.mdx) to some other colour (which can be transparent).
