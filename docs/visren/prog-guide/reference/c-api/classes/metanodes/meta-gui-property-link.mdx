# Meta_GUIPropertyLink

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

A GUIPropertyLink is a simple way of connecting [Meta_Button](../metanodes/meta-button.mdx), [Meta_Slider](../metanodes/meta-slider.mdx), and [Meta_ScrollBar](../metanodes/meta-scroll-bar.mdx) toggle / change events to other node properties without using a [Meta_EventHandler](../metanodes/meta-event-handler.mdx). This is a two-way link. 

`#include <meta_gui.hpp>`
