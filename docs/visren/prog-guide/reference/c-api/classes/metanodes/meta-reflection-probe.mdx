## Detailed Description

```cpp
struct Meta_ReflectionProbe;
```

A ReflectionProbe represents a local skybox providing dynamic reflections of the surrounding environment. 

These are positioned in the scene by adding them as a child of an [Assembly](../metanodes/meta-assembly.mdx) node and transforming that.
