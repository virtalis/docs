# Keyboard Shortcuts

## General

| Shortcut          | Description                                                                                                                        |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------|
| Ctrl + N          | New scene.                                                                                                                         |
| Ctrl + O          | Open scene.                                                                                                                        |
| Ctrl + Shift + O  | Merge scene.                                                                                                                       |
| Ctrl + I          | Import file.                                                                                                                       |
| Ctrl + S          | Save scene.                                                                                                                        |
| Ctrl + Shift + S  | Save scene as.                                                                                                                     |
| Ctrl + P          | Show/hide the Properties window.                                                                                                   |
| Ctrl + U          | Show/hide the View Widget.                                                                                                         |
| Ctrl + L          | Show/hide the Script Editor.                                                                                                       |
| Ctrl + D          | Show/hide the Diagnostics window.                                                                                                  |
| Ctrl + Q          | Show/hide the Animation Sequencer.                                                                                                 |
| Ctrl + K          | Show/hide the Gradient Editor.                                                                                                     |
| Ctrl + B          | Show/hide the Review window.                                                                                                       |
| Ctrl + J          | Show/hide the Network window.                                                                                                      |
| Ctrl + Shift + F  | Show/hide the Search window.                                                                                                       |
| Ctrl + G          | Show/hide the Gallery.                                                                                                             |
| Ctrl + Z/Y        | Undo/Redo.                                                                                                                         |
| Ctrl + X/C/V      | Cut/Copy/Paste.                                                                                                                    |
| Delete            | Delete.                                                                                                                            |
| PrtScn            | Capture the screen as an image, image sequence or video (saved to **Documents**\\**VisionaryRender**\\**Screenshots** by default). |
| Ctrl + H          | Enable/disable headlight.                                                                                                          |
| Ctrl + M          | Enable/disable mouse movement with the tracked hand controller.                                                                    |
| Alt + M           | Enable/disable a mirror of the current viewport (useful when configuring tracking).                                                |
| Ctrl + W          | Enable/disable wireframe mode.                                                                                                     |
| Ctrl + Tab        | Cycle tabs in a tabbed window.                                                                                                     |
| Alt               | Click through the GUI into 3D scene.                                                                                               |
| Alt + Z           | Show/hide the GUI.                                                                                                                 |
| Alt + Right-click | Open immersive context menu (in 3D scene).                                                                                         |
| Ctrl + 1          | Enable/disable region selection in the 3D scene.                                                                                   |
| Ctrl + 2          | Enable/disable the measuring tool.                                                                                                 |
| Ctrl + 3          | Enable/disable the annotation tool.                                                                                                |
| F1                | Show the compiled HTML version of this manual.                                                                                     |
| F2                | Rename selection.                                                                                                                  |
| F3                | Enable/disable selection.                                                                                                          |
| F4                | Enable/disable the snap home tool.                                                                                                 |
| Ctrl + F3         | Enable/disable selection and children.                                                                                             |
| F6                | Show/hide Settings window.                                                                                                         |
| F9                | Enable/disable PRS Manipulator.                                                                                                    |
| F10               | Enable/disable stereo.                                                                                                             |
| F11               | Cycle full screen modes.                                                                                                           |
| F12               | Enable/disable tracking.                                                                                                           |
| Esc               | Close the current dialog if you have one open.                                                                                     |
| Alt + F4          | Exit the application.                                                                                                              |

:::note
When using Copy and Paste in the tree views, Paste will automatically select the newly created assembly. Pasting the same object multiple times pastes as sibling to the last pasted object, instead of a child. This action can also span multiple selections within different tree branches.
:::

## 3D Scene

| Shortcut                         | Description                                                                                     |
|----------------------------------|-------------------------------------------------------------------------------------------------|
| Alt + Left-click object          | Fly to.                                                                                         |
| Shift + Left-click object        | Select GeoGroup, Mesh, and Material in Library tree.                                            |
| Ctrl + Shift + Left-click object | Select Material in Library tree.                                                                |
| Drag-and-drop from object        | Drag Material onto another object or GUI Frame/Drag into Script Editor to create variables/etc. |

## File Browsers

| Shortcut          | Description                                  |
|-------------------|----------------------------------------------|
| Ctrl + Shift + N  | Create a new folder.                         |
| Arrow Keys        | Navigate to previous/next file in directory. |
| Alt + Left Arrow  | Navigate to previous directory in history.   |
| Alt + Right Arrow | Navigate to next directory in history.       |
| Home/End          | Go to first/last file in directory.          |

## Text Edit Boxes

| Shortcut        | Description                         |
|-----------------|-------------------------------------|
| Ctrl + A        | Select all.                         |
| Ctrl + Insert   | Copy.                               |
| Shift + Insert  | Paste.                              |
| Backspace       | Delete.                             |
| Home/End        | Go to first/last character in line. |
| Ctrl + Home/End | Go to first/last line.              |
| Page Up/Down    | Go to previous/next page.           |

## Tree Views

| Shortcut              | Description                                         |
|-----------------------|-----------------------------------------------------|
| Ctrl + Left-click     | Multi select.                                       |
| Shift + Left-click    | Range select.                                       |
| Up/Down Arrow         | Go to previous/next item in tree.                   |
| Left/Right Arrow      | Collapse/expand current item or go up/down a level. |
| Page Up/Down          | Go to previous/next page.                           |
| + or - (Numpad)       | Expand or collapse current item.                    |
| * or / (Numpad)       | Expand or collapse current item and children.       |
| Enter                 | Activate selection.                                 |
| Esc                   | Clear selection.                                    |
| Ctrl + A              | Select all.                                         |
| Ctrl + F              | Quick search.                                       |
| Shift + Up/Down Arrow | Range select.                                       |

## Quick Search

| Shortcut      | Description                        |
|---------------|------------------------------------|
| Enter         | Go to next search result.          |
| Up/Down Arrow | Go to previous/next search result. |
| Esc           | Cancel search/clear search text.   |

## Script Editor

| Shortcut     | Description                              |
|--------------|------------------------------------------|
| F7           | Compile current script.                  |
| Ctrl + F7    | Compile all scripts.                     |
| Ctrl + E     | Execute current script (in console tab). |
| Ctrl + F4    | Close current script.                    |
| Ctrl + F     | Show the find and replace toolbar.       |
| Tab          | Indent highlighted code.                 |
| Shift + Tab  | Un-indent highlighted code.              |
| Ctrl + Space | Automatically indent highlighted code.   |

## Animation Sequencer

| Shortcut                  | Description                                 |
|---------------------------|---------------------------------------------|
| Ctrl + Left-click         | Select multiple animations or frames.       |
| Shift + Left-click        | Select a rectangle of animations or frames. |
| Middle-click              | Pan timeline.                               |
| Ctrl + Mouse wheel or +/- | Zoom in/out.                                |
| Home/End                  | Seek to start/end of sequence.              |
| Page Up/Down              | Pan timeline to previous/next page.         |
| Up/Left Arrow             | Seek to previous frame.                     |
| Down/Right Arrow          | Seek to next frame.                         |
