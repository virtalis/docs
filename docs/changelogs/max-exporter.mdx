# Max Exporter

## 1.09

*Released 2023-09-26*

#### Added

- FUS-3194 - Adds support for 3ds Max 2024.

#### Removed

- FUS-3194 - Removes support for 3ds Max 2014-2016.

---

## 1.07

*Released 2022-11-02*

#### Added

- FUS-2787 - Adds support for 3ds Max 2023.

---

## 1.06

*Released 2022-01-24*

#### Added

- FUS-1944 - Adds support for 3ds Max 2021 and 2022.

#### Changed

- FUS-2167 - Improves window layouts on high-resolution displays.

---

## 1.05

*Released 2020-03-17*

#### Added

- FUS-1060 - Adds a warning when skin vertices are affected by more than 4 bones.
- FUS-1165 - Adds support for the subdivisions and sides options on splines, and sets exported splines to tube mode when they have non-zero thickness.
- FUS-1258 - Adds support for Spline Select and Linked XForm modifiers affecting spline vertices.

---

## 1.04

*Released 2019-12-06*

#### Added

- FUS-4 - Adds support for exporting visibility tracks as sequence property tracks.
- FUS-5 - Adds support for spline IK control modifier to animate splines.
- FUS-423 - Adds support for 3ds Max 2020.
- FUS-1053 - Adds tooltips to the settings.

#### Changed

- FUS-253 - Rebrands the user manual.
- FUS-392 - Improves the GUI layout and updates the logo.

#### Fixed

- FUS-1048 - Fixes version number mismatch and broken revision number.

---

## 1.03

*Released 2019-08-09*

#### Fixed

- DEV-1848 - Fixes bug that caused Look At constraints to be improperly translated.
- DEV-2119 - Fixes bug where the exporter would complain about a missing DLL when 3ds Max starts. (FD273, FD318)
