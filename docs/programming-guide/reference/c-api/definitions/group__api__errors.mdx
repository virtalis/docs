# Error Codes

Error codes that might be set if an API function fails, assuming the log level is something other than VRTREE_ERRORLEVEL_NONE. 

## Attributes

|                | Name           |
| -------------- | -------------- |
| const uint32_t | **[VRTREE_API_INVALID_HANDLE](../definitions/group__api__errors.mdx#vrtree-api-invalid-handle)** <br />API function expected a valid object handle.  |
| const uint32_t | **[VRTREE_API_INVALID_METANODE](../definitions/group__api__errors.mdx#vrtree-api-invalid-metanode)** <br />API function expected a valid metanode name/handle.  |
| const uint32_t | **[VRTREE_API_INVALID_PARAMETER](../definitions/group__api__errors.mdx#vrtree-api-invalid-parameter)** <br />API function expected a valid parameter (buffer, string, etc)  |
| const uint32_t | **[VRTREE_API_INVALID_PROPERTY](../definitions/group__api__errors.mdx#vrtree-api-invalid-property)** <br />API function expected a valid property name/index.  |
| const uint32_t | **[VRTREE_API_INVALID_SECURITYCONTEXT](../definitions/group__api__errors.mdx#vrtree-api-invalid-securitycontext)** <br />API function was called without adequate permissions.  |
| const uint32_t | **[VRTREE_API_MISSING_MIGRATIONS](../definitions/group__api__errors.mdx#vrtree-api-missing-migrations)** <br />API function attempted to perform migrations but some are missing.  |
| const uint32_t | **[VRTREE_API_NOT_ALLOWED](../definitions/group__api__errors.mdx#vrtree-api-not-allowed)** <br />API function attempted to do something inherently forbidden.  |
| const uint32_t | **[VRTREE_API_OK](../definitions/group__api__errors.mdx#vrtree-api-ok)** <br />No error - operation succeeded.  |
