## Detailed Description

```cpp
struct Meta_VertexPrim;
```

A VertexPrim specifies a range of vertices in [Meta_VertexAttrib](../Classes/struct_meta___vertex_attrib.mdx) siblings to render from, requiring this data to be in primitive order (and possibly duplicated) 

See [Meta_IndexPrim](../Classes/struct_meta___index_prim.mdx) for a list of supported prim type names.
