## Detailed Description

```cpp
struct Meta_AnimationAction;
```

Animation keyframes for bool, int, float, double and world float properties. 

An action frame holds a script that can be executed by animation sequences

Valid child types: [Meta_EventHandler](../Classes/struct_meta___event_handler.mdx), [Meta_AudioBuffer](../Classes/struct_meta___audio_buffer.mdx), [Meta_AudioSound](../Classes/struct_meta___audio_sound.mdx)
