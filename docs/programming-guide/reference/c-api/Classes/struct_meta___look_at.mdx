## Detailed Description

```cpp
struct Meta_LookAt;
```

A LookAt node contains settings and child [Meta_LookAtTarget nodes, required when a [Meta_Assembly](../Classes/struct_meta___assembly.mdx) contraint type is set to LookAt. 

This is expected to be a child of a [Meta_Assembly](../Classes/struct_meta___assembly.mdx).
