## Detailed Description

```cpp
struct Meta_AudioBuffer;
```

An AudioBuffer represents sound data that can be played by multiple sources. 

This node is usually created in a library and referenced from [Meta_AudioSound](../Classes/struct_meta___audio_sound.mdx) or [Meta_AudioTrack](../Classes/struct_meta___audio_track.mdx) nodes. These are positional.
