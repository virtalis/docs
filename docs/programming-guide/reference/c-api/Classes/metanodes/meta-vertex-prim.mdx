## Detailed Description

```cpp
struct Meta_VertexPrim;
```

A VertexPrim specifies a range of vertices in [Meta_VertexAttrib](../metanodes/meta-vertex-attrib.mdx) siblings to render from, requiring this data to be in primitive order (and possibly duplicated) 

See [Meta_IndexPrim](../metanodes/meta-index-prim.mdx) for a list of supported prim type names.
