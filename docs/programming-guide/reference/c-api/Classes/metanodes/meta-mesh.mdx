## Detailed Description

```cpp
struct Meta_Mesh;
```

A Mesh is a collection of vertices and lists which can be rendered as a polyhedral object. 

Each Mesh node contains data about the mesh stored in [Meta_VertexAttrib](../metanodes/meta-vertex-attrib.mdx) and [Meta_IndexPrim](../metanodes/meta-index-prim.mdx) nodes; vertices, indices, normals, texture coordinates, etc.
