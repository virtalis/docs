# Meta_Dimension

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

A Dimension references two [Meta_MeasuringPoint](../metanodes/meta-measuring-point.mdx) nodes and calculates the distances between them. It also provides scene visuals depicting the dimensions.
