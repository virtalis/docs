## Detailed Description

```cpp
struct Meta_AudioTrack;
```

An AudioTrack provides full control over playback and looping of data in a [Meta_AudioBuffer](../metanodes/meta-audio-buffer.mdx). 

One sound source is allocated to this and is retained to prevent AudioSound interrupting it.
