# Meta_GUI

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

A GUI is a container for other GUI elements such as buttons and stacks. It controls how the GUI is drawn.
