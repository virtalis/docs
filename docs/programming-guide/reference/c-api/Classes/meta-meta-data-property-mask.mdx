## Detailed Description

```cpp
struct Meta_MetaDataPropertyMask;
```

A MetaDataPropertyMask contains is a 256-bit value to store bits corresponding to up to 256 properties of a node. 

This is primarily used by groups and snapshots to specify which properties in a snapshot are to be applied. The [vrBitsToInteger](../Modules/group__api__lua.mdx#vrbitstointeger) Lua function can be used to create an integer from up to 32 bit values.
