## Detailed Description

```cpp
struct Meta_Viewpoint;
```

[Meta_Viewpoint represents camera settings for a specific view on the scene. It should be a child of a [Meta_Assembly](../Classes/struct_meta___assembly.mdx). 

This is attached to an Assembly whose world transform is used as a camera position.
