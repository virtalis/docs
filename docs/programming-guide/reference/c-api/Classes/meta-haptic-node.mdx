# Meta_HapticNode

**Module:** **[Metanodes](../Modules/group__api__metanodes.mdx)** **/** **[Metanode Structure Definitions](../Modules/group__api__metadefs.mdx)**



A HapticNode is added to [Meta_Assembly](../Classes/struct_meta___assembly.mdx) much the same way as [Meta_Collision](../Classes/struct_meta___collision.mdx), and allows the assembly to be felt by haptic devices. 


`#include <meta_haptic.hpp>`
