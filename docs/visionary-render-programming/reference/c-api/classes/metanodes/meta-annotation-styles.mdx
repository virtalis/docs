# Meta_AnnotationStyles

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

AnnotationStyles is a container for GUI style nodes, applied to annotations. These are usually [Meta_ModelMarkerStyle](../metanodes/meta-model-marker-style.mdx) or similar and are saved with the scene so are customisable. 

`#include <meta_annotation.hpp>`
