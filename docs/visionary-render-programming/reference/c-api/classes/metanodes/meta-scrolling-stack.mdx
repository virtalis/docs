# Meta_ScrollingStack

**Module:** **[Metanodes](../../modules/metanodes/index.mdx)** **/** **[Definitions](../../modules/metanodes/definitions.mdx)**

A ScrollingStack is a stack that adds scrollbars if its contents are larger than its size. It renders its contents to a texture like [Meta_BufferedStack](../metanodes/meta-buffered-stack.mdx). 

`#include <meta_gui.hpp>`
